# 공가지도 발표자료 소스 체크리스트

발표자료는 직접 제작한다. 아래 자료를 확보한 뒤 PowerPoint 또는 미리캔버스에 넣는다.

## 서비스 캡처

- [ ] 공가지도 랜딩 또는 메인 화면 캡처
  - 사용 페이지: 1, 6
  - 확보 위치: 최종 배포 URL `/`
- [ ] 공가지도 지도 화면 캡처
  - 사용 페이지: 1, 6
  - 확보 위치: 최종 배포 URL `/map`
- [ ] 후보 마커가 보이는 지도 화면 캡처
  - 사용 페이지: 6
  - 확보 위치: `/map`
- [ ] 후보 상세 패널 또는 상세 페이지 캡처
  - 사용 페이지: 6, 10
  - 확보 위치: `/house/H00018` 등 샘플 후보 상세
- [ ] 우선조사 추천 또는 위험도 표시 캡처
  - 사용 페이지: 6, 9
  - 확보 위치: 후보 상세 또는 대시보드
- [ ] 지자체 대시보드 전체 화면 캡처
  - 사용 페이지: 6, 10
  - 확보 위치: `/dashboard`
- [ ] 관리 우선순위 Top 10 화면 캡처
  - 사용 페이지: 9, 10
  - 확보 위치: `/dashboard`

## 통계 자료

- [ ] 2024년 미거주 주택 160만 호 / 8.0% 통계
  - 사용 페이지: 2
  - 권장 출처: 대한민국 정책브리핑, 2024년 인구주택총조사 결과
  - URL: https://www.korea.kr/briefing/policyBriefingView.do?newsId=156721680
- [ ] 법적 빈집 관리 대상 약 13만 호 통계
  - 사용 페이지: 2, 3
  - 권장 출처: 국토연구원 빈집 실태 파악 및 관리체계 관련 브리프
  - URL: https://www.krihs.re.kr/boardDownload.es?bid=0008&list_no=398058&seq=1
- [ ] 빈집 정비 또는 실태조사 제도 자료
  - 사용 페이지: 3, 9
  - 권장 출처: 국토교통부, 행정안전부, 지자체 공식 자료
- [ ] 공공데이터 활용 경진대회 심사/대회 정보
  - 사용 페이지: 1, 10
  - 권장 출처: 국가교통 데이터 오픈마켓
  - URL: https://www.bigdata-transportation.kr/pageant/dashboard/CMPE_000000000020042

## 브랜드/마무리 자료

- [ ] 공가지도 로고 PNG 또는 워드마크
  - 사용 페이지: 1, 11
  - 없으면 텍스트 워드마크로 제작
- [ ] 서비스 URL
  - 사용 페이지: 1, 11
  - 최종 URL: `https://kyo-map.vercel.app` 또는 최신 배포 URL
- [ ] QR코드
  - 사용 페이지: 11
  - 생성 후 휴대폰 스캔 테스트 필수
- [ ] 발표자 이름
  - 사용 페이지: 1
- [ ] 발표자 이메일
  - 사용 페이지: 11
- [ ] 대회명
  - 사용 페이지: 1

## 최종 확인

- [ ] 모든 숫자에 출처가 붙어 있다.
- [ ] `빈집 확정`, `공식 판정`, `정확도 99%` 표현이 없다.
- [ ] `빈집 후보`, `AI 추정`, `우선조사 추천` 표현을 사용했다.
- [ ] 서비스 화면 캡처가 흐릿하지 않다.
- [ ] QR코드가 실제 URL로 연결된다.
- [ ] 발표자료에는 PPT 자동 생성물이나 웹 슬라이드 캡처가 섞이지 않았다.
