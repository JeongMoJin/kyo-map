# Hashtag Buckets — 해시태그 5 버킷 전략

총 15~25개를 아래 5 버킷에서 고르게 뽑아 한·영 믹스.

## 버킷 1 — 브랜드·제품 (3~5개)

카드뉴스가 다루는 **고유명사**. 제품·회사·서비스 이름.

**예시 (Claude Design 주제)**:
`#ClaudeDesign #클로드디자인 #Anthropic #앤트로픽 #ClaudeOpus4_7`

**예시 (다른 주제)**:
- 애플 이벤트: `#AppleEvent #애플이벤트 #iPhone #아이폰 #WWDC`
- OpenAI 뉴스: `#OpenAI #GPT #ChatGPT #SamAltman #DALLE`

**규칙**: 한글 표기 + 영문 표기 쌍으로 넣으면 검색 커버리지 2배.

## 버킷 2 — 경쟁·업계 (3~5개)

같은 시장 또는 대체재 관련 태그. 비교·대체 검색에 걸림.

**예시 (디자인 툴 관련)**:
`#Figma #피그마 #피그마대체 #Canva #캔바 #디자인툴 #AI디자인툴`

**예시 (AI 코딩 관련)**:
`#Cursor #GitHubCopilot #Replit #AI코딩 #코드에디터`

**규칙**: 최소 1개는 "대체" / "alternative" 키워드 포함 — 대체 솔루션을 찾는 검색자 공략.

## 버킷 3 — 일반 검색 (4~6개)

AI·테크·디자인 전반의 대형 태그. 도달량 가장 큼.

**예시**:
`#AI #AI뉴스 #테크뉴스 #AI트렌드 #ITNEWS #techtrends #AItools #artificialintelligence #tech #innovation`

**규칙**: 대형 태그 2~3개 + 중형 태그 2~3개 섞기. 초대형만 쓰면 경쟁 치열로 노출 약함.

## 버킷 4 — 커뮤니티·방법론 (2~4개)

해당 분야 작업자들이 모이는 니치 태그. 저장율·댓글 유도.

**예시 (디자인 계열)**:
`#디자인시스템 #UXUI #프로덕트디자인 #designsystem #productdesign`

**예시 (AI 계열)**:
`#프롬프트엔지니어링 #PromptEngineering #LLM #GenAI #AIresearch`

**규칙**: 타깃 독자가 자발적으로 follow하는 태그. 도달은 작지만 **저장율이 높음**.

## 버킷 5 — 브랜드 고유 (1~2개)

**우리 계정의 시그니처 태그**. 일관되게 매번 붙여 계정 브랜딩.

**예시**:
- `@promptwAIve` 계정 → `#promptwAIve #프롬프트웨이브`

**규칙**: 고유 태그는 **우리 계정 게시물만** 담는 필터 역할. 팔로워가 이 태그로 과거 글 찾아보기 쉬움.

## 조합 예시 (총 20개)

**Claude Design 카드뉴스 캡션에 실제로 사용된 해시태그**:

```
#클로드디자인        ← 버킷 1 (브랜드 한글)
#ClaudeDesign       ← 버킷 1 (브랜드 영문)
#앤트로픽           ← 버킷 1
#Anthropic          ← 버킷 1
#AI디자인           ← 버킷 2 (업계)
#AI디자인툴         ← 버킷 2
#프로토타입         ← 버킷 2
#Figma              ← 버킷 2 (경쟁)
#피그마대체         ← 버킷 2 (대체 키워드)
#디자인툴           ← 버킷 2
#AI툴               ← 버킷 3 (일반)
#ChatGPT            ← 버킷 3
#클로드             ← 버킷 3
#ClaudeAI           ← 버킷 3
#AI뉴스             ← 버킷 3
#테크뉴스           ← 버킷 3
#AI트렌드           ← 버킷 3
#디자인시스템       ← 버킷 4 (커뮤니티)
#프롬프트엔지니어링  ← 버킷 4
#promptwAIve        ← 버킷 5 (브랜드 고유)
```

## 금지·주의

- 같은 태그 반복 금지 (`#AI #AI #AI`)
- 의미 없는 범용 태그 금지 (`#일상 #좋아요반사 #맞팔`)
- 인스타 스팸 필터 걸리는 태그 금지 (follow4follow 류)
- 띄어쓰기 포함된 태그는 작동 안 함 (`#AI 뉴스` → X, `#AI뉴스` → O)

## 최신 태그 트렌드 확인

해시태그 집행 전, 인스타 앱에서 검색해 **최근 24시간 인기 태그**를 훑어 보강 가능. AI 분야는 트렌드가 주 단위로 바뀜.
