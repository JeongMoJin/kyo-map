# Tone Variants — 캡션 톤 3종

카드뉴스 톤에 맞춰 캡션 어조·이모지·종결을 조정. 기본은 **brand**.

## news — 속보·사건·폭로·이슈

**이모지**: ⚡ 🔥 📉 📈 🚨 📌
**종결**: 단정체 ("~했다", "~공개됐다", "~밝혀졌다")
**리듬**: 짧은 문장, 구체적 수치, 출처 언급

**예시 훅**:
- "⚡ 속보: [사건·발표 한 문장]"
- "🔥 [놀랄만한 수치] — 오늘 공개"
- "📉 [하락/경쟁사 변화] 하루만에 [수치]"

**예시 본문**:
```
⚡ 속보: 앤트로픽, Claude Design 공개

2026년 4월 17일, Figma 주가 7% 하락. 디자인 AI 경쟁이 다시 불붙었다.

📌 핵심 팩트
· 발표: 앤트로픽 랩 (공식 블로그)
· 엔진: Claude Opus 4.7
· 대상: Pro · Max · Team · Enterprise

⚡ 가장 놀라운 수치
복잡한 페이지 프로토타입, 다른 툴은 20+ 프롬프트 필요.
Claude Design은 2개로 끝. 앤트로픽 공식 발표.
```

**적합 주제**: 출시·발표·사건·폭로·주가·인물 이슈

## brand — 브랜드·라이프스타일·제품 소개 (기본 추천)

**이모지**: 🎨 ✨ 💡 🚀 📌 ☕
**종결**: 존댓말 + 부드러운 단정체 ("~합니다", "~였습니다", "~할 수 있어요")
**리듬**: 리드미컬, 짧지만 따뜻함, 독자와 대화하듯

**예시 훅**:
- "🎨 [브랜드]가 이제 [새 시도]한다고?"
- "✨ 60년 전 시작된 [브랜드], 지금은 [수치]"
- "💡 매일 마시는 [제품], 그 뒤에 있는 [이야기]"

**예시 본문**:
```
🎨 클로드가 디자인까지 한다고?

2026년 4월 17일, 앤트로픽 랩이 Claude Design을 공개했습니다.
코드 한 줄 없이 프로토타입과 슬라이드를 만드는 AI 툴.

📌 이번 발표 핵심
· 엔진: Claude Opus 4.7
· 입력 소스: 텍스트, 이미지, 문서
· 내보내기: Canva · PDF · PPTX · HTML
```

**적합 주제**: 제품·브랜드·커피·뷰티·여행·라이프스타일

## editorial — 에디토리얼·감성·회고

**이모지**: 💡 🌿 📝 ☕ 🕰️
**종결**: 담백 서술체 ("~이다", "~했다") 또는 반말 서술
**리듬**: 느긋한 문장, 한 호흡, 긴 한 문장

**예시 훅**:
- "💡 어떤 기술은 도구가 아니라, 태도다"
- "🕰️ 60년이 지나도 변하지 않은 것들"
- "📝 디자인의 미래는, 결국 대화로 돌아간다"

**예시 본문**:
```
💡 어떤 기술은 도구가 아니라, 태도다.

2026년 4월 17일, 앤트로픽은 Claude Design을 세상에 내놨다.
코드 없이도 형태를 짓는 일이 가능해진 하루.

📌 바뀐 것
· 시간: 하루에서 분 단위로
· 언어: 코드에서 대화로  
· 경계: 디자이너와 기획자 사이가 흐려졌다

⚡ 20에서 2로
프롬프트 20번이, 단 2번으로 압축됐다.
손끝 작업이 생각의 작업으로 옮겨갔다는 뜻.
```

**적합 주제**: 에세이형 주제·회고·브랜드 철학·담론

## 톤 감지 힌트

카드뉴스의 `.hook-title` 이나 `.q-text` 의 어조에서 감지:

- 강렬한 동사·숫자 강조 (`공개`, `하락`, `충격`) → **news**
- 따뜻한 반어·감탄 (`~이라고?`, `~나요?`) → **brand**
- 담담한 명사 마침표 → **editorial**

불명확하면 **brand** (기본).

## 해시태그 톤 매칭

- **news**: `#AI뉴스 #테크뉴스 #속보` 비율 ↑
- **brand**: `#브랜드스토리 #AI트렌드` 비율 ↑
- **editorial**: `#에세이 #테크에세이 #디자인철학` 추가

## CTA 톤 매칭

```
news → ✅ 공유해서 주변 디자이너에게 알려주세요
brand → ✅ 이 소식이 유익했다면 저장·공유
editorial → ✅ 이 글이 생각을 불러일으켰다면, 한번 더 읽어보세요
```
