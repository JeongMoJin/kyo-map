# Instagram Caption Generator

## 언제 쓰나

- 카드뉴스 (`hyperframes-card-news` 스킬로 만든 토픽) 완성 후 인스타 업로드를 위한 본문 캡션이 필요할 때
- 사용자가 "인스타 본문", "캡션 써줘", "인스타 글", "IG 캐러셀 설명" 등으로 요청할 때
- 16:9 슬라이드덱이어도 인스타에 올리려는 경우 같은 스킬로 생성 가능

## 결과물

**파일**: `topics/<주제>/instagram-caption-ko.txt` (한국어 기본) 또는 `-en.txt` (영문)

**구조** (위에서 아래):

```
[🎨 훅 1줄]       ← 스크롤을 멈추게 하는 질문·선언 (25자 이내)

[리드 2줄]         ← 날짜/누가/한 줄 요약 (훅 포함 총 ~125자 내 권장)

📌 [핵심 섹션 라벨]
· 불릿 1            ← "·" interpunct 사용, 각 불릿 15자 내외
· 불릿 2
· 불릿 3
(· 불릿 4-5 — 콘텐츠에 따라)

⚡ [바이럴 스탯 / 임팩트 포인트]
  가장 공유될 만한 수치·사건 1개를 2-3줄로 풀어 씀

📉 [시장·영향 맥락]      ← 있으면 추가, 없으면 생략
  경쟁사 반응·업계 영향 한 줄

🚀 [CTA 링크/접근 방법]
  URL 또는 접근 경로

---

[마감 코멘트 한 줄]        ← 타깃 독자에게 한 번 더 어필

✅ 저장·공유 유도
✅ @계정 핸들 + 팔로우 유도

.
.
.

#해시태그 #해시태그 #... (15~25개, 5개 버킷 믹스)
```

## 작성 규칙

### 1. 125자 훅 규칙

인스타 앱에서 "더 보기" 버튼이 뜨기 전 첫 125자 정도만 노출된다. 여기에 **훅 1줄 + 날짜/누가 + 한 줄 요약**이 들어가야 스크롤 중에 읽힌다.

좋은 예:
```
🎨 클로드가 디자인까지 한다고?

2026년 4월 17일, 앤트로픽 랩이 Claude Design을 공개했습니다.
```
→ 정확히 ~80자. "더 보기" 전에 전부 노출.

### 2. 훅 공식

| 패턴 | 예시 |
|------|------|
| 질문형 | "클로드가 디자인까지 한다고?" |
| 충격 선언 | "Figma 주가, 하루 만에 7% 하락" |
| 리스트 예고 | "2026년 디자이너가 꼭 알아야 할 5가지" |
| 숫자 시작 | "20 → 2. 프롬프트 수가 10분의 1로" |

훅 앞 이모지: 🎨(디자인) / ⚡(속보) / 🔥(핫) / 📌(정리) / 🚀(출시) / 💡(인사이트) / 📉(하락) / 📈(성장) 중 내용 맞춰 1개.

### 3. 불릿 작성

- 반드시 `·` (middle dot, interpunct) — `-` 나 `•` 보다 깔끔함
- 각 불릿 15자 내외, 6개 넘지 않게
- 각 불릿 시작을 **카테고리 라벨 + 콜론** 형태로 일관되게:
  ```
  · 엔진: Claude Opus 4.7
  · 입력 소스: 텍스트, 이미지, 문서
  · 내보내기: Canva · PDF · HTML
  ```

### 4. 바이럴 스탯 (⚡ 섹션)

덱에서 가장 임팩트 있는 숫자·사건 1개를 골라 2-3줄로 풀어 쓴다. 이 섹션이 게시물 저장률을 결정.

예: "20 → 2" 프롬프트 감소 / "Figma -7%" 주가 하락 / "1시간 → 5분" 작업 단축

### 5. 시장 맥락 (📉 선택)

경쟁사·업계 반응이 있으면 1줄 추가. 없으면 생략.

### 6. CTA (🚀)

링크 형식은 플레인 URL. 인스타는 본문 링크가 클릭 안 되므로 **"프로필 링크 참고"** 또는 **"바이오 링크"** 안내를 덧붙이면 좋다.

### 7. 저장·팔로우 유도 (✅)

```
✅ 이 소식이 유익했다면 저장·공유
✅ 더 빠른 AI·디자인 트렌드는 @promptwAIve
```

계정 핸들은 해당 토픽의 `photo-cover` `.hook-handle` 에 들어간 브랜드 핸들과 일치시킨다.

### 8. 해시태그 구조

**5개 버킷에서 고르게** (자세한 리스트 → `references/hashtag-buckets.md`):

1. 브랜드·제품 (3-5개): `#ClaudeDesign #Anthropic #클로드디자인 #앤트로픽`
2. 경쟁·업계 (3-5개): `#Figma #피그마대체 #디자인툴 #AI디자인툴`
3. 일반 검색 (4-6개): `#AI뉴스 #테크뉴스 #AI트렌드 #AI`
4. 커뮤니티·방법론 (2-4개): `#프롬프트엔지니어링 #디자인시스템`
5. 브랜드 고유 (1-2개): `#promptwAIve`

**총 15~25개**. 20개 전후가 도달률·노출 균형점. 한·영 믹스 필수 (외국인 검색·AI 검색 모두 대비).

### 9. 공백·"." 트릭

본문 끝 ~ 해시태그 시작 사이에 `.` 만 있는 줄 3-5개를 둔다:

```
.
.
.

#해시태그...
```

이유: 인스타 앱이 해시태그를 **리포스트 스팸으로 덜 보이게** 하고, 해시태그가 앞 본문 읽기를 방해하지 않게 시각적으로 떨어뜨림.

## 톤 variants

세부 구성은 `references/tone-variants.md` 참조.

- **news** — 속보·사건·폭로형 (⚡🔥📉 이모지, 단정 종결)
- **brand** — 브랜드·라이프스타일 (🎨✨ 이모지, 존댓말)
- **editorial** — 에디토리얼·감성 (💡🌿 이모지, 담백 서술체)

기본 추천은 **brand**. 뉴스성 주제면 **news**.

## 흐름

1. **입력**: 완성된 `topics/<주제>/index.html` (카드 내용 파악 대상)
2. **추출**: 각 카드에서 핵심 문장 뽑아내기 (`.hero-title`, `.stat-num`, `.feature-body p`, `.q-text` 등)
3. **매핑**: 9단계 구조의 각 섹션에 카드 내용 배치
4. **작성**: 아래 템플릿에 채움, 125자 훅 규칙 확인, 해시태그 5 버킷 믹스
5. **저장**: `topics/<주제>/instagram-caption-ko.txt`
6. **보고**: 글자 수 (Instagram 2,200자 한도 내), 첫 125자 확인, 해시태그 수 확인

## 검증 체크리스트

작성 후 확인:
- [ ] 글자 수: 2,200자 이하 (`wc -c <file>`)
- [ ] 첫 125자 내에 훅 + 리드 포함 (`head -c 125 <file>`)
- [ ] 불릿은 `·` 사용 (중점)
- [ ] 해시태그 15-25개
- [ ] 해시태그 5 버킷 전부 커버 (브랜드·경쟁·일반·커뮤니티·고유)
- [ ] 브랜드 핸들이 카드 `.hook-handle` 과 일치
- [ ] 본문-해시태그 사이 "." 줄 3-5개
- [ ] CTA 링크 포함 (있을 때)
