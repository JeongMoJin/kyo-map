# Other Transitions

## 언제 쓰나
- 다른 카테고리에 안 맞는 실험적/독특한 감각이 필요할 때
- 덱 전체에 한 번만 쓰는 포인트 트랜지션
- **톤**: 상황별
- **추천 duration**: 0.5–0.9초

## 설치
```bash
npx hyperframes add transitions-other
```
→ `compositions/transitions-other.html`.

## 와이어링
```html
<div data-composition-src="compositions/transitions-other.html"
     data-start="4.5" data-duration="0.7"
     data-track-index="1"
     data-width="1080" data-height="1350">
</div>
```

## 파라미터
- 쇼케이스 내부 변형 선택

## 짝
- 덱당 1회만 — 포인트용
